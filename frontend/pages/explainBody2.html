1. 지갑 만들기

  1. Metamask 가입하기

    2. Metamask는 이더리움 블록체인과 연결된 지갑입니다.

    1) Chrome에서 Metamask extension 설치하기

    메타마스크는 Chrome 브라우저를 통해 쉽게 사용할 수 있습니다.
    Chrome 브라우저가 설치되어 있지 않다면, 여기에서 다운로드 받아 설치하시기 바랍니다. Chrome Extension에서 Metamask를 찾아 설치를 시작합니다.

    2) Metamask 계정 만들기

    Metamask 설치가 완료되면, Chrome 브라우저의 주소창 오른쪽에 다음과 같은 여우모양 버튼 생기게 됩니다.
    Metamask가 필요할 때 여우 버튼을 클릭하면 지갑에 로그인하여 사용할 수 있습니다.
    버튼을 눌러 들어가게되면, 처음에 아래와 같은 화면이 나타납니다. 사용할 비밀번호를 입력하면, 계정이 생성되게 됩니다.

  2. **니모닉 코드 보관 법

    3. 니모닉 코드는 다른 컴퓨터에서 Metamask의 계정이 필요할 때 계정을 복구하기 위해 꼭 필요합니다.
    우리가 보통 사용하는 아이디와 비슷한 개념입니다.
    다만 좀 더 어렵게 만들어 놓은 것이죠.
    니모닉 코드와 비밀번호가 유출되면 계좌에 있는 모든 돈을 잃게될 수 있으니 안전한 장소에 보관해야 합니다.
    12개의 니모닉 코드 단어들은 종이에 적어두고, 많은 사람들이 사용하는 컴퓨터에 저장하지 않는 것을 추천합니다.

  3. 개인 키 위치

    4. 니모닉 코드까지 다 저장하고 계정을 생성하게 되면, 모든 과정이 끝난 것입니다.
    한 계좌에서 다른 계좌로 이더와 같은 암호화화폐를 보낼 때는 계좌의 주소가 필요합니다.
    이와 같은 주소는 아래와 같은 위치에서 확인할 수 있습니다.

  4. 개인 키의 용도

    5. 개인 키는 이더리움 블록체인에 차용증을 등록하고, 서명하는 데에 꼭 필요합니다.
    우리의 주민등록증, 운전면허증과 같이 모든 개인키는 다르기 때문에 사람을 구분하는 데 사용할 수도 있습니다.
    여기서는 채무자와 채권자가 누구인지 확인하기 위해 사용합니다.
